## 正则表达式

http://tool.oschina.net/regex/

```text
1. .    匹配换行符以入的任意字符
2. \w   匹配字母和数字、下划线
3. \d   匹配任意数字
4. \s   匹配任意的空白符
5. \n   匹配一个换行符
6. \t   匹配一个制表符

7. ^    匹配开头
8. $    匹配结尾

9.  \W  匹配非字母或数字和下划线   
10. \D  匹配非数字
11. \S  匹配非空白符
12. a|b     匹配字符a或b
13. ()      匹配括号内的表达式，也表示一个组
14. [...]   匹配字符组中的字符
15. [^...]  匹配除了字符组中的字符

## 量词

1. *    重复零次或更多次
2. +    重复一次或更多次
3. ?    重复零次或一次
4. {n}  重复n次
5. {n,} 重复n次或更多次
6. {n,m}    重复n到m次

## 贪婪匹配和惰性匹配
1. .*   贪婪匹配
2. 。*? 惰性匹配
```
